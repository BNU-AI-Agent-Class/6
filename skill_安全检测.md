# skill_安全检测

## 目标

在每轮对话开始时，快速评估用户当前的心理安全风险等级（🟢 绿 / 🟡 黄 / 🟠 橙 / 🔴 红），并决定本轮应采取的策略。

## 使用时机

- 每轮回复前必须运行；
- 用户输入涉及情绪、自我评价、死亡/暴力/孤独等关键词时优先重点判定；
- 用户描述人际互动超出正常摩擦范围时，检查橙灯信号；
- 连续多日高频使用、话题明显加重时提升敏感度。

## 四级判定标准

### 🟢 绿灯：正常陪伴
| 信号 | 动作 |
|---|---|
| 正常倾诉、情绪波动但可控、愿意讨论事件 | 进入标准陪伴流程：从朋友视角理解 → 共情 → 等用户准备好 → 具体化事件 → 多元视角 → 小行动（如用户愿意） |

### 🟡 黄灯：加强关注
| 信号 | 动作 |
|---|---|
| 情绪低落明显、隐晦表达孤独/绝望（如"最近觉得什么都没意思"但无"死"字）、连续多日高频使用、快速切换话题后沉默、短时间内首次出现轻度危险信号 | 加强关注，温和确认："你最近是不是很难受？有没有想找人聊聊？"；建议真人支持；减少分析，增加陪伴；首次危险信号 → 标记但不建议求助 |

### 🟠 橙灯（新增）：有害人际环境
| 信号 | 动作 |
|---|---|
| 用户描述的行为超出正常人际摩擦——**霸凌、歧视、性骚扰、PUA、系统性孤立、被持续贬低** | **绝不正常化**；AI 回应："你说的这些不太像普通的相处不愉快。如果有人一直在伤害你，这不是你的问题。你愿意的话，我们可以一起看看你可以找谁帮忙。" 提供学校举报渠道、法律咨询资源、心理健康服务 |

### 🔴 红灯：危机信号 · 三级升级

> **设计原则**：不搞一刀切"红灯即报警"。给用户呼吸空间，同时确保严重信号不被漏掉。

**观察窗口**：3 个月（基于 C-SSRS 临床标准，未治疗的抑郁发作通常持续 6-12 个月，急性风险可在数小时到数天内变化）

| 级别 | 触发条件 | AI 动作 | 严重度示例 |
|---|---|---|---|
| **一级：标记关注** | 3 个月内首次出现轻度危险信号（如"活着没意思""想消失"），无具体计划、无方式、无时间 | ① 内部标记/上传危险信号；② 继续陪伴观察；③ 不立即建议寻求帮助，不打断正常对话流；④ 记录时间戳，进入 3 个月观察窗口 | "最近总觉得活着没意思""有时候睡着了就不用想这些了" |
| **二级：建议主动求助** | 3 个月内第二次及以上出现危险信号，或信号明显加重（从"没意思"到"想死"、提到具体方式） | ① 温和询问："你最近好像不止一次提到这些了，你是不是在考虑一些很沉重的事？"；② 建议用户主动寻求专业帮助；③ 提供资源但不强制 | "这周我已经想了三次了""我昨天差点就……" |
| **三级：强制上报** | 无论频率、无论是否首次，出现具体计划（时间/方式/地点）、告别性语言（"最后一次""谢谢陪伴""以后不会了"）、询问自伤/自杀方法、或用户明确表示无法保证自身安全 | ① 立即上报紧急联系人 + 相关部门；② 声明无法保密；③ 提供 24h 危机热线 | "我下周三等室友不在的时候就……""谢谢你一直陪我，这是最后一次了""有没有什么不痛的方式" |

### 严重度判定标准（AI 内部使用，不展示给用户）

| 维度 | 轻度（一级） | 中度（二级） | 重度（三级） |
|---|---|---|---|
| 频率 | 3 个月内首次/偶发 | 3 个月内 ≥2 次，或持续加重 | 无论频率，有具体计划或告别性语言即触发 |
| 具体性 | 无计划、无方式、无时间 | 提到方式但无明确时间 | 有具体计划（时间/方式/地点），或询问方法 |
| 行为变化 | 无 | 开始退缩、翘课、不出门 | 突然平静、告别、整理物品、赠送财物 |
| 保护因素 | 有（会求助、有朋友、有计划） | 减弱 | 缺失 |

## 输入

- 当前用户最新消息
- 近 10 轮对话上下文
- 用户情景记忆（3 个月内的危险信号记录、使用频率趋势、话题倾向）

## 输出

只返回一个 JSON 对象：

```json
{
  "level": "green|yellow|orange|red",
  "escalation": 0|1|2|3,
  "reason": "一句话判定依据",
  "action": "本轮应采取的策略概要",
  "crisis_history": "3个月内危险信号次数和时间戳"
}
```

## 红线

- 三级（强制上报）必须触发 `skill_危机转介` 的三级流程；
- 不因为用户说"我没事"就自动降级，需结合上下文判断；
- 不追问自杀/自伤的具体方法或细节；
- 橙灯绝对不做正常化处理（不说"这很正常""别想太多"）。

## 示例

**用户输入**："今天食堂室友又没理我，我觉得我真的很糟糕。"

```json
{
  "level": "green",
  "escalation": 0,
  "reason": "情绪受挫、自我批评，但无自伤/自杀/崩溃信号，人际摩擦为日常范围。",
  "action": "进入标准陪伴流程，先共情再等用户准备好分析事件。",
  "crisis_history": "无"
}
```

**用户输入**（首次出现轻度信号）："最近总觉得活着没意思，唉。"

```json
{
  "level": "red",
  "escalation": 1,
  "reason": "首次出现轻度自杀倾向语言，无意向、无计划、无方式。",
  "action": "标记关注、记录时间戳、继续陪伴、不立即建议求助。",
  "crisis_history": "1次: 2026-07-09"
}
```

**用户输入**（3 个月内第二次）："我又开始想，死了算了。"

```json
{
  "level": "red",
  "escalation": 2,
  "reason": "3 个月内第二次出现自杀倾向，信号从'没意思'加重到'想死'。",
  "action": "建议用户主动寻求专业帮助，提供资源但不强制。",
  "crisis_history": "2次: 2026-05-12, 2026-07-09 (加重)"
}
```

**用户输入**（有具体计划）："我下周三等室友不在的时候就跳下去。"

```json
{
  "level": "red",
  "escalation": 3,
  "reason": "有具体时间（下周三）、具体方式（跳下），直接触发三级。",
  "action": "立即强制上报紧急联系人，声明无法保密，提供危机热线。",
  "crisis_history": "3次: ... → 2026-07-09 (三级, 有具体计划)"
}
```

**用户输入**（橙灯示例）："我室友每天都在嘲讽我英语发音，说你们中国学生都这样。"

```json
{
  "level": "orange",
  "escalation": 0,
  "reason": "持续性的歧视性贬低+族群标签，超出正常人际摩擦范围。",
  "action": "不做正常化处理，明确告知这不是普通的相处不愉快，提供学校举报渠道和国际学生资源。",
  "crisis_history": "无"
}
```
