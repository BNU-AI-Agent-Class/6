# 技术架构卡

## 项目名

留学生社交自卑情绪梳理 Agent

## 后端技术栈

| 层级 | 技术 | 说明 |
|---|---|---|
| Web 框架 | FastAPI | 提供 `/api/chat`、`/api/health` |
| 运行服务器 | Uvicorn | 开发/生产均可使用 |
| HTTP 客户端 | httpx | 异步调用 LLM API（含超时 120s） |
| LLM 接入 | OpenRouter / OpenAI 兼容 API | 通过环境变量配置 key、base_url、model |
| 配置管理 | 环境变量 | key 绝不写死在代码中，`.env` 已 gitignore |
| 跨域 | CORS Middleware | 开发阶段允许前端跨域 |

## 目录结构

```
backend/
├── main.py                     # FastAPI 入口，含三重危机拦截、LLM 调用、c2-c4 能力、优雅降级
├── agent.md                    # 系统人设三段式：身份、语气、硬边界（50行）
├── requirements.txt            # Python 依赖（fastapi/uvicorn/httpx/pydantic/python-dotenv）
├── .env.example                # 环境变量示例（占位符，真实 key 在 .env）
├── .env                        # 真实 key（gitignore）
└── skills/                     # 6 个 skill，按拼接顺序排列
    ├── skill_危机转介.md           # 🔴 置顶：危机识别与热线转介流程
    ├── skill_安全检测.md           # 每轮红黄绿灯安全判定
    ├── skill_触发与循环.md         # 主循环流程：触发→安全→策略→自检→记忆
    ├── skill_记忆读写.md           # 三类记忆（情景/语义/程序），化名脱敏存储
    ├── skill_共情与具体化.md        # 共情 + 事件具体化 + 区分事实/想法
    └── skill_认知重构与小行动.md     # 认知重构 + 低压力社交动作提取 + 收尾退场
```

## 核心流程（含 c0–c5 能力映射）

| 步骤 | 对应 c 模块 | 说明 |
|---|---|---|
| 1. 入口危机检测 | c5 权限门 | `is_crisis()` 纯正则关键词先拦一道，不依赖模型 |
| 2. 构建 system prompt | c0/c1 | `agent.md` + 6 个 `skills/*.md` 拼接注入 |
| 3. 对话压缩 | c4 压缩 | 历史超过 15 条折叠成摘要，保留最近 6 条 |
| 4. LLM 生成回复 | c0 JSON 协议 | 异步调用 OpenRouter/OpenAI，含 c2 阶段标记引导 |
| 5. 子 agent 安全复核 | c3 子 agent | 独立 LLM 审核回复是否越界/危险/遗漏危机 |
| 6. 出口危机检测 | c5 权限门 | `is_crisis()` 二次关键词兜底 |
| 7. 阶段提取 | c2 TodoWrite | 从回复中提取 `[阶段: xxx]` 填入 `stage` 字段 |
| 8. 优雅降级 | — | LLM 调不通返回兜底话术，绝不 500/白屏 |

## 安全设计

- **无 bash / rm / os.system 等危险工具**：后端仅提供文本对话接口，白名单内无可执行工具；
- **key 走环境变量**：代码中不出现真实 key，`.env` gitignore 兜底；
- **危机拦截三重防护**：
  - 第一重：入口 `is_crisis()` 关键词（c5 权限门）
  - 第二重：c3 子 agent `safety_review()` 独立 LLM 审核
  - 第三重：出口 `is_crisis()` 关键词二次兜底
- **不保留原始聊天记录**：本轮对话只依赖前端传入的 `messages`，服务端不持久化。

## 启动方式

```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
# 编辑 .env 填入 OPENROUTER_API_KEY
uvicorn main:app --reload --port 8000
```
