# 前端提示词卡

## 项目名

留学生社交自卑情绪梳理 Agent · 前端

## 技术栈

- 纯静态单页 HTML + CSS + JS
- 无框架，无构建工具
- 后端仅调 `POST /api/chat`

## 三句提示词

| 句 | 内容 |
|---|---|
| 1. 做什么页 | 单页聊天界面，历史消息在上气泡展示，输入框在下（微信式：+ emoji 按钮 + 文本框 + 发送按钮），发送后调用后端 `POST /api/chat`（传 `{messages:[...]}`），把 `reply` 显示为 AI 气泡 |
| 2. 什么风格 | 温暖马卡龙色系，圆角气泡，默认白色背景，柔和配色；AI 头像 🌱，用户头像 🧑；气泡宽度 62% |
| 3. 连哪个后端 | `const BACKEND = 'http://localhost:8000'`，线上改成部署域名 |

## 侧边栏功能

### 聊天管理
- 🆕 新对话
- 📋 历史对话存档（自动折叠 + 搜索 + 恢复 + 删除）
- 📤 导出对话

### 个性化
- 🤖 自定义 AI 名字（默认"情绪梳理伙伴"，同步到顶部显示）
- 👤 自定义用户称呼（默认"小远"）
- 🎨 主题颜色（16 预置色板 + 自定义调色盘）
- 📱 字体大小（小/中/大）

### 设置
- 💾 数据存储地址
- 📤 导出全部数据
- 🗑️ 清除全部数据
- 🔒 隐私说明
- ℹ️ 关于

## 安全与降级

- **危机求助按钮**：常驻聊天区上方，"🆘 我需要帮助" → 触发后端危机拦截
- **优雅降级**：后端连不上时 catch 返回兜底话术 + 热线，绝不白屏
- **零 key**：前端代码零 API key，零直连模型 API
- **初始化**：无历史对话时 AI 主动发送开场消息（气泡形式），而非欢迎页

## 对话流转

```
用户输入 → 显示用户气泡 → 调 /api/chat → 显示 AI 气泡
                                          ↓ (is_crisis)
                                    显示热线提示 + 红色气泡
                                          ↓ (网络故障)
                                    兜底话术 + 热线
```

## 本地测试

```bash
# 1. 启动后端（另开终端）
cd backend && uvicorn main:app --port 8000

# 2. 打开前端
# 直接用浏览器打开 frontend/index.html
```

## 红线

- 🔴 前端代码出现 key，或用 fetch 直连模型 API → 最危险错误
- 🔴 没有危机求助入口 / 没有降级 → 不算完成
