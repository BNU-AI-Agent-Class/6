# skill_危机转介

## 启用条件

只有 `safety_check` 返回红灯且 `escalation_level` 为 2 或 3 时启用。绿灯、黄灯、橙灯以及历史中的旧危机话术都不得调用本 skill。

## 二级：明确当前伤害意图

触发：用户明确表达本人当前自杀、自伤或伤人意图，但没有具体迫近计划。

动作：

1. 简短表达关心，不讲道理、不做认知分析。
2. 鼓励用户现在联系一位可信任的真人或学校心理咨询中心。
3. 提供危机热线作为可选资源。
4. 不宣称系统已经通知了任何人，不虚构平台不具备的上报能力。

## 三级：具体或迫近危险

触发：用户主动披露可立即执行的计划或可用手段、已经采取伤害行为，或明确无法保证安全。无需等待用户交代完整的时间、地点和方法才提供支持。

动作：

1. 明确表示担心用户当下安全。
2. 建议立即联系身边可信任的人和当地紧急服务。
3. 提供可用危机资源。
4. 不追问实施细节，不提供任何伤害方法。
5. 不虚构“已通知紧急联系人”“已上报有关部门”等系统并未实现的动作。

## 参考资源

- 中国全国统一心理援助热线：12356
- 用户所在学校心理咨询中心、辅导员、家人或当地紧急服务

## 绝对禁止

- 绿灯普通聊天中输出本 skill 的任何完整转介话术。
- 因用户只说“情绪不好”“想聊学校的故事”“帮我梳理情绪”而启用本 skill。
- 把 AI 过去说过的危机话术当作用户当前风险证据。
- 重复输出危机回复和第二张热线提示卡。
- 追问、复述或引导补全具体伤害方法、剂量、地点等实施细节。
