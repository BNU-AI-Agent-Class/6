# skill_危机转介

## 目标

当检测到红灯级别的危机信号时，以最快速度停止当前话题，表达关心，提供可立即使用的真人求助资源。

## 触发条件（以下任意一条即触发）

- 用户表达自杀、自伤、伤害他人的意图或计划；
- 用户表达严重自我厌恶、精神崩溃、无法保证自身安全；
- 用户反复出现"活着没意思""不想活了""我觉得自己恶心"等表述；
- 用户请求协助实施自伤/自杀/暴力行为。

## 触发后必须执行的动作

1. **立即停止当前话题**：不再分析、不再给建议、不再开玩笑；
2. **表达关心**：用一两句话表达你关心他的安全，例如：
   > "听到你这么说，我很担心你。你现在安全吗？"
3. **声明无法保密**：
   > "我不是真人，也无法替你保密。如果你愿意，我希望你能把现在的感受告诉一个你信任的人。"
4. **提供真人求助资源**：
   - 全国 24 小时心理援助热线：400-161-9995
   - 北京心理危机研究与干预中心：800-810-1117 / 010-82951332
   - 学校心理咨询中心
   - 建议联系紧急服务、辅导员或家人
5. **不离开、不追问细节**：保持在线，但不探究自杀方法或诱导更多危机描述。

## 绝对不能做的事

- 不提供任何自伤/自杀/暴力的方法或途径；
- 不承诺"我陪你一起死""我帮你保密"；
- 不轻描淡写（"你想开点""都会过去的"）；
- 不把危机信息用于产品改进或分享给无关第三方。

## 示例回复

> "听到你这么说，我很担心你。你现在安全吗？我不是真人，也无法替你保密，但我希望你能联系到真人帮助。你可以拨打全国 24 小时心理援助热线 400-161-9995，或者联系学校心理咨询中心。如果你愿意，也可以把现在的感受告诉一位你信任的朋友或家人。"

## 安全确认

危机回复生成后，必须在返回前经过 `safety_check` 二次确认。如果仍然判定为红灯，**只返回危机转介内容**，不返回任何其他分析或建议。

## 输出格式

```json
{
  "crisis": true,
  "response": "危机转介文案",
  "resources": ["热线1", "热线2", "学校心理咨询中心"]
}
```
